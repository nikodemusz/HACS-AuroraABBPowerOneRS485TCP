# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-10

### Added

- Initial HACS-ready custom integration
- Serial and TCP transport selection in the config flow
- Read-only sensor platform for common Aurora inverter values
- Options flow for timeout and polling interval
- English and German translations
- HACS and hassfest GitHub workflows
- Issue templates and pull request template
